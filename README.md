# Pixelpart for Godot 4

This repository contains the official Pixelpart plugin for Godot 4.3+. The plugin allows you to play Pixelpart effects directly within the Godot engine.

## Installation

### Asset Library

The plugin is available in the official Godot Asset Library. Download and install the *Pixelpart* plugin in the Asset Library tab in the Godot editor, then activate it in the project settings.

### Manual Install

You can also manually install the plugin by downloading it from [pixelpart.net](https://pixelpart.net/plugins/). Extract the archive and move the *pixelpart* folder into the *addons* folder below the root folder of your Godot project. If no *addons* folder exists, create one. Lastly, start the Godot editor and activate the plugin in the project settings.

## Usage

A detailed usage guide can be found [here](https://pixelpart.net/documentation/godot4/) on the Pixelpart website. The plugin also offers an extensive scripting API available for modifying effects dynamically via GDScript.

## Supported Platforms

The plugin can be used on the following platforms:

- Windows
- Linux
- macOS
- iOS
- Android
- Web
